━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

给谁用
  香港公司的老板 / 会计 / 审计师
  用来统计一个审计账期内的银行进账流水
  输出 Excel 给会计师做报价、做账、出财报

怎么用 · 三步
  1. 双击「流水统计器.html」或「一键打开.command」
     → 用 Chrome / Edge / Safari 打开

  2. 登录 DBS 星展网银 → 下载每月的「综合月结单 PDF」
     (注意：必须是银行原版 PDF，不能是手机拍照 / 截图)

  3. 把 PDF / 文件夹 / ZIP 压缩包整个拖进页面
     → 工具自动挑出 PDF，其他自动忽略
     → 顶部显示「解析完成」后点「下载 Excel 报表」

功能要点
  · 支持一次性拖入整个 ZIP 压缩包、整个文件夹、多个 PDF
  · 非月结单（Word / Excel / 图片等）自动忽略
  · 处理过程树状显示，每个文件实时看到状态
  · 右上角可开关提示音

Excel 报表 · 4 个 sheet
  1) 进账汇总 · 月 × 币种 矩阵，报价按这个算
  2) 进账明细 · 每笔进账的日期 / 对手方 / 金额 / 源文件
  3) 全部交易 · 含支出，用于和银行余额做完整性核对
  4) 源文件   · 哪张 PDF 覆盖了哪些月份

安全说明
  · 所有 PDF 只在本机浏览器里解析
  · 不会上传到任何服务器
  · 关掉页面所有数据即清除
  · 首次打开需联网（下载解析库），之后可断网使用

支持银行
  · DBS 星展银行（中国 / 香港）综合月结单 Combined Statement

  其他银行（HSBC / 恒生 / 招行永隆 / 中银香港 / 汇丰）
  暂未适配，如有需要请联系你的会计师。

常见问题
  Q: 从微信 / iCloud / 网盘里的文件拖不进来？
  A: 先右键另存到本地桌面，然后再拖；或点击灰框用文件选择器。

  Q: 显示「没能解析出来」？
  A: 99% 是 PDF 格式问题。必须是银行原版 PDF，
     不能是手机截图 / 拍照 / 扫描件。

  Q: 缺了某个月怎么办？
  A: 不影响使用，工具只统计你提供的月份。
     补齐后重新拖一次即可。

  Q: 汇率 / 账期想自己指定？
  A: 页面「进阶选项」里全部可改。
